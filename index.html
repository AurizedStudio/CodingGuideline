文字コード

原則 UTF-8。

改行コード

原則 CR+LF(Windows)。

DOCTYPE

基本的にはHTML5。

もしくは、XHTML1.0 Transitional または HTML4.01 Transitional。

バリデート

W3Cが勧告するHTML5の仕様に準拠します。
以下のバリデータでエラー無しを目指します。
http://validator.w3.org/

改行とインデント

ソースの改行は、ブロック要素の終了タグの後ろ等、適宜行います。
body内の要素には、構造に合わせて適切にインデントします。
インデントには半角スペース４つ分を使用。

パスの記述

原則としてサイト内リンク、画像リンク共に相対パス（ドキュメント）。

<title>タグ

トップページのタイトルには、サイト名としてサイトや会社の名前を載せることに加え、会社の所在地、主要な取扱商品などの重要な情報を含める。
２階層目以降は、基本的にはページ名、サイト名の順に「 | 」（半角スペース＋半角縦棒＋半角スペース）で区切ります。
例：ページ名 | サイト名

<meta＞タグのkeywords

キーワード毎に半角カンマ「,」で区切ります。

<hX>タグ

<h1>は特に指定なければ、Topページではサイトタイトル、ロゴなどに使用し、その他のページではページ固有のタイトルにあたるテキスト or img要素に対して使用します。

<img>タグ

意味のある画像は、原則背景ではなくimg要素として配置します。（文書を補完する画像、単独で内容を説明している画像、メインイメージなど）
alt属性にて画像の代替となるテキストを設定します。（特に指定がない場合は、その画像の簡易説明をaltの内容とします。電話でその画像の説明をする感じです。装飾用の画像など、相応しい代替テキストが存在しない画像の場合には「alt=""」とします）

<div>タグ

ソースの内容を把握しやすいようにするために、class、id付きの閉じ</div>タグの後ろに適宜コメントを入れます。
例：
</div>
<!-- / #header -->

<input>、<textarea>、<select>タグ

id属性に対してスタイルを適用しません。id属性は変更されることを考慮して、class属性を使用してスタイルを適用します。

チェックボックスとラジオボタン

<label for="任意値">と対応したid属性を必須とします。
例：<label for="example01"><input type="checkbox" id="example01" />ラベル</label>

フォント、フォントファミリー

単位は「px」または「em」を使用します。

指定無ければ次のフォントで記述します。
font-family: Verdana, Roboto, "Droid Sans", "游ゴシック", YuGothic, "ヒラギノ角ゴ ProN W3", "Hiragino Kaku Gothic ProN", "メイリオ", Meiryo, sans-serif;

カラーコード

小文字で３桁表示可能な場合は３桁で記述します。
例：#fff, #f89, #5a5b5c

id名、class名

文書構造上、意味のある名前にします。
必要な場合は必要な数のハイフン「-」を使用し、末尾に２桁の連番を付けます。
連番の中で詳細がある場合はハイフン「-」に続けて記載します。
例：
wrapper（全体を囲む枠）
wrapper-inner（全体を囲む枠の内側枠）
wrapper-outer （全体を囲む枠の外側枠）
header（ヘッダー）
utility（メインのナビではない項目のメニューなど）
g-nav（グローバルナビゲーション）
gnav（グローバルナビゲーション）
sub-nav（サブナビゲーション）
subnav（サブナビゲーション）
content（コンテンツ枠）
main（メインコンテンツ枠）
sub（サブコンテンツ枠）
copyright （著作項目要素）
footer（フッター）
list-basic01, list-basic02, list-basic03
icon01, icon02, icon03
tableselect01, tableselect02, tableselect03, tableselect03-inner, tableselect04

決めているclass名

ボタン等の状態を変化させるときのクラス名："○○--hoge"
js等で状態変化させるときのクラス名："is-fuga"
ある要素の外側要素："その要素名-outer"
ある要素の内側要素："その要素名-inner"

ショートハンドプロパティ

marginやpaddingはショートハンドを使います。（個別の継承を考えなくてもよいため）
例：margin-bottom: 2px; -> margin: 0 0 2px 0;

画像ファイル名

識別名+「-」+「ページ名orカテゴリ名orセクション名」+「-」+「詳細」+連番２桁.拡張子
（「ページ名orカテゴリ名orセクション名」、「詳細」は不要の場合省略します。連番は２つ以上ファイルが存在する場合に付けます。）
例：
title-topic01.gif
bg-main-table01.png
